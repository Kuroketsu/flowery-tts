flowery_tts

Flowery TTS | API Wrapper

A free Text-to-Speech service providing 850+ voices with English translation support. This package is an unofficial wrapper of Flowery TTS API for the Dart programming language.

Features

  • Complete endpoint coverage.
  • Fully documented.
  • Supports in all platforms.
  • Class-based.
